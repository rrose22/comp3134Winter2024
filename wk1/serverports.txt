A server port is a numerical endpoint that facilitates communication between software applications on a server and client devices; 

the server port range spans from 0 to 65535, and the number represents 16 bits; 

common server ports include 80 for HTTP, 443 for HTTPS, 21 for FTP, 22 for SSH, 25 for SMTP, 110 for POP3, 143 for IMAP, 53 for DNS, 23 for Telnet, and 22 for SFTP.
